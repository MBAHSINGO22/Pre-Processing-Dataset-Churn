<h2>📌 Tentang Project</h2>

<p>
Project ini berfokus pada proses <b>data preprocessing</b> menggunakan Python dan Jupyter Notebook.
Preprocessing dilakukan untuk mempersiapkan dataset agar lebih bersih, terstruktur, dan siap digunakan dalam proses analisis data maupun machine learning.
</p>

<p>
Tahapan preprocessing sangat penting karena kualitas data akan mempengaruhi performa model analisis dan machine learning yang dihasilkan.
</p>

<hr>

<h2>🎯 Tujuan Project</h2>

<ul>
  <li>Membersihkan data dari nilai yang tidak valid</li>
  <li>Menangani missing value</li>
  <li>Melakukan transformasi data</li>
  <li>Mempersiapkan dataset untuk machine learning</li>
  <li>Meningkatkan kualitas dan konsistensi data</li>
</ul>
